# NewsBoardAI Project Plan

## Core Idea

NewsBoardAI is a personal news intelligence dashboard. The user searches a topic, and the app fetches a small number of recent news results, analyzes them, and displays a compact visual summary.

## MVP Scope

The first version should:

- Accept a search query
- Fetch top 3 to 5 news results
- Extract title, source, date, snippet, and URL
- Classify sentiment using a simple model
- Detect basic event types using rules or a small classifier
- Show a compact dashboard result

## Initial Use Cases

- Company or stock news
- Video game news
- Sports team or athlete news
- Political or public issue news
- Product launch or tech news

## Design Goal

The result should feel like a small dashboard, not a long article summary.

The output should include:

- Overall signal
- Sentiment meter
- Event tags
- Key summary
- Top sources
- Possible impact
- Confidence level
